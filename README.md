# Flowz Viewer — distribution

This repo holds the **built** Flowz Viewer bundle, served by GitHub Pages at
**https://flowzhq.github.io/viewer/**.

The viewer renders Flowz system views straight out of a `.flowz-artifacts`
repo in your GitHub org — sign in with your own GitHub credential; nothing
leaves your browser except calls to `api.github.com`. Try it without an
account: [`#demo`](https://flowzhq.github.io/viewer/#demo).

Deep links:

```text
https://flowzhq.github.io/viewer/#<system>
https://flowzhq.github.io/viewer/#system=<system>&repo=<owner>/.flowz-artifacts&branch=branch-<name>
```

There is no source here — only minified build output, deployed automatically
from a private repo. Commit history is the deploy log. Issues and feedback:
contact the Flowz team.
